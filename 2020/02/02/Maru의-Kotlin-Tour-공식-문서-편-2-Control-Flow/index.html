If Expression

if는 expression, 즉 값을 return 하므로 tenary operator(condition ? then : else)가 없다고 한다.(난 tenary operator 좋은데..)

그럼 if 표현식을 보자

마지막 코드인 val max = if (a > b) a else b가 tenary operator와 비슷해보인다.

if문을 이런 식으로 써서 변수에 값을 넣어 줄 수도 있다.
